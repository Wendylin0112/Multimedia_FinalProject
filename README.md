# Multimedia_FinalProject
我們的專案旨在為音訊檔案生成相對應的影片檔，幫助人們視覺化一段美妙的音訊。 目前可以識別雨聲、溪流聲、海浪聲、火車、寫作、蟲鳴鳥叫的聲音，並由ai模型生成一段短影片。 我們希望未來可以實現將夢幻音樂聲生成看得見的場景，增加娛樂新體驗。
